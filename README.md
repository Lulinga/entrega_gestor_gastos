Proyecto Final Programación Web Gestor de gastos

Montuori, Olmos

Idea general:

El proyecto se basa en crear un gestor de gastos, donde cada usuario pueda llevar sus cuentas como en un Libro Diario. Que puedan anotar sus gastos y ver un balance a final de mes. O añadir un objetivo de dinero para realizar alguna compra y calcular un monto conveniente para ahorrar en base a los gastos fijos o dinero disponible cada mes. Que cuente con un calendario para recordar pagos importantes o fechas límites.

Objetivos:

Proporcionar una herramienta a las personas para que puedan manejar de manera eficiente su dinero. Que sea fácil de utilizar, clara y ordenada. Proteger los datos y la privacidad de los usuarios. Que sea lo más precisa posible. Llegar a ser recomendada y bien valorada por los usuarios.

Arquetipo de usuarios:

Dirigido a personas que tengan algún tipo de emprendimiento, para que lleven sus gastos organizadamente. Para uso personal, contribuyendo al ahorro.

Plataformas:

Página web (disponible en cualquier tipo de dispositivo). Aplicación de celular. Aplicación de escritorio (siendo más simple de usar).
